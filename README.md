# App-M: Mars Kolonisi Simülasyonu

Mars'ta bir koloni kurma ve yönetme temalı, 2D mikro yönetim strateji oyunu.

## Özellikler
- 2D Pixel Art / Vektörel Görseller
- Kaynak Yönetimi (Oksijen, Su, Enerji, Yiyecek)
- Bina İnşa Etme ve Geliştirme
- Kolonist Yönetimi
- Oyun İçi Satın Alımlar (IAP)

## Geliştirme Ortamı
- **Motor:** Godot Engine 4.x
- **Dil:** GDScript
- **Platformlar:** iOS, Android
