# 🌀 Mars Tycoon - Anomaliler (Anomalies) Spesifikasyonları

Bu dosya, Mars yüzeyinde beliren gizemli sinyallerin ve keşif süreçlerinin detaylarını içerir.

## 📊 Anomali Mekaniği
*   **Belirme:** Haritada rastgele, boş karelerde periyodik olarak belirirler.
*   **Görsel:** Mavi/Turkuaz parlayan sinyal efekti.
*   **Maliyet:** 50 Araştırma Puanı (🔬).
*   **Süre:** 20 saniye tarama süresi.

## 🎁 Ödüller (Loot Table)
| Olasılık | Ödül Tipi | Açıklama |
| :--- | :--- | :--- |
| **30%** | **Breakthrough** | Rastgele bir 'Çığır Açan Keşif' kilidi açılır. |
| **40%** | **Resources** | Kaynak paketi (15 İzotop, 50 Metal, 1000 Kredi). |
| **30%** | **Data Boost** | 200-500 Araştırma Puanı kazandırır. |
